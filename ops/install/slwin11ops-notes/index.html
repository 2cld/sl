    local slwin11ops (DHCP - see note below)

    ⚠ CONFIRM: LAN IP observed as both 192.168.0.143 and 192.168.0.141 on different dates. This machine uses DHCP (no static reservation). The IP will drift unless a MAC reservation is added on the router for MAC B0-83-FE-65-80-80. Recommend adding a DHCP reservation on 192.168.0.1 to lock this down.
